# hierarchical-diagram-studio

A desktop application for designing hierarchical graph-based diagrams with linked subdiagrams, manual layout control, and fully customizable node connections.

![logo](https://raw.githubusercontent.com/trucomanx/HierarchicalDiagramStudio/main/screenshot.png)

## 1. Installing

To install the package from [PyPI](https://pypi.org/project/hierarchical_diagram_studio/), follow the instructions below:


```bash
pip install --upgrade hierarchical_diagram_studio
```

Execute `which hierarchical-diagram-studio` to see where it was installed, probably in `/home/USERNAME/.local/bin/hierarchical-diagram-studio`.

### Using

To start, use the command below:

```bash
hierarchical-diagram-studio
```
