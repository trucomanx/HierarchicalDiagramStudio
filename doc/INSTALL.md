# hierarchical-diagram-studio

A desktop application for designing hierarchical graph-based diagrams with linked subdiagrams, manual layout control, and fully customizable node connections.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/hierarchical_diagram_studio/

```bash
pip install --upgrade hierarchical_diagram_studio
```

Using:

```bash
hierarchical-diagram-studio
```

## Install from source
Installing `hierarchical-diagram-studio` program

```bash
git clone https://github.com/trucomanx/HierarchicalDiagramStudio.git
cd HierarchicalDiagramStudio
pip install -r requirements.txt
cd src
python -m build
pip install dist/hierarchical_diagram_studio-*.tar.gz
```
Using:

```bash
hierarchical-diagram-studio
```

## Uninstall

```bash
pip uninstall hierarchical_diagram_studio
```
